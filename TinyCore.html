TinyCore Linux

TinyCore is Linux is a minimal Linux operating system focusing on providing
a base system using Busybox and FLTK.
TinyCore was developed by Robert Shingledecker.
The distribution is notable for its small size (11 to 16 MB) and minimalism; additional functions are provided by extensions.
Tiny Core Linux is free and open source software and is licensed under the GNU General Public License version 2.
